### I. General (Across Most Applications)

These are the fundamental shortcuts you'll use constantly.

*   **Copy:** `Ctrl + C` (`Cmd + C`)
    *   Copies selected text or items to the clipboard.
*   **Cut:** `Ctrl + X` (`Cmd + X`)
    *   Cuts selected text or items to the clipboard (removes them from their original location).
*   **Paste:** `Ctrl + V` (`Cmd + V`)
    *   Pastes content from the clipboard.
*   **Undo:** `Ctrl + Z` (`Cmd + Z`)
    *   Reverses the last action. Indispensable for mistakes!
*   **Redo:** `Ctrl + Y` (`Cmd + Shift + Z`)
    *   Reverses an Undo action (brings back what you undid).
*   **Select All:** `Ctrl + A` (`Cmd + A`)
    *   Selects all items or text in the current window or document.
*   **Save:** `Ctrl + S` (`Cmd + S`)
    *   Saves the current document or file.
*   **Open:** `Ctrl + O` (`Cmd + O`)
    *   Opens an existing document or file.
*   **New:** `Ctrl + N` (`Cmd + N`)
    *   Creates a new document, window, or file.
*   **Print:** `Ctrl + P` (`Cmd + P`)
    *   Opens the print dialog for the current document.
*   **Find:** `Ctrl + F` (`Cmd + F`)
    *   Opens a search bar to find text within the current document or page.
*   **Close Current Window/Tab:** `Ctrl + W` (`Cmd + W`)
    *   Closes the active window or browser tab.
*   **Close Application:** `Alt + F4` (`Cmd + Q`)
    *   Closes the active application.
*   **Switch Applications/Windows:** `Alt + Tab` (`Cmd + Tab`)
    *   Cycles through open applications. Release keys to select.
*   **Minimize Window:** `Win + M` (`Cmd + M`)
    *   Minimizes the active window to the taskbar/dock.
*   **Screenshot (Full Screen):** `PrtSc` (Print Screen key) or `Win + PrtSc` (Windows saves to Screenshots folder) (`Cmd + Shift + 3`)
    *   Captures the entire screen.
*   **Screenshot (Partial/Selection):** `Win + Shift + S` (`Cmd + Shift + 4`)
    *   Allows you to select a specific area of the screen to capture.
*   **Screenshot (Specific Window - Windows):** `Alt + PrtSc`
    *   Captures only the active window.
*   **Refresh/Reload:** `F5` or `Ctrl + R` (`Cmd + R`)
    *   Reloads the current page or refreshes the window content.
*   **Rename Item:** `F2` (on selected file/folder) (`Enter` on selected file/folder, then type)
    *   Allows you to rename a selected file or folder.
*   **Open Help:** `F1` (`Cmd + ?` or `Help` menu)
    *   Opens the help documentation for the current application.
*   **Toggle Full Screen:** `F11` (`Cmd + Ctrl + F`)
    *   Makes the active window go full screen or exit full screen.
*   **Lock Screen:** `Win + L` (`Cmd + Ctrl + Q`)
    *   Locks your computer, requiring a password to unlock.
*   **Open Task Manager (Windows) / Force Quit (macOS):** `Ctrl + Shift + Esc` or `Ctrl + Alt + Del` (`Cmd + Option + Esc`)
    *   Opens Task Manager (Windows) to manage processes or Force Quit Applications (macOS).

### II. Text Editing & Navigation

These shortcuts are incredibly useful when working with documents, emails, or any text field.

*   **Bold:** `Ctrl + B` (`Cmd + B`)
    *   Applies or removes bold formatting.
*   **Italic:** `Ctrl + I` (`Cmd + I`)
    *   Applies or removes italic formatting.
*   **Underline:** `Ctrl + U` (`Cmd + U`)
    *   Applies or removes underline formatting.
*   **Move Cursor One Word Left/Right:** `Ctrl + Left/Right Arrow` (`Option + Left/Right Arrow`)
    *   Moves the cursor a whole word at a time.
*   **Select One Word Left/Right:** `Ctrl + Shift + Left/Right Arrow` (`Option + Shift + Left/Right Arrow`)
    *   Selects text one word at a time.
*   **Move Cursor to Beginning/End of Line:** `Home` / `End` (`Cmd + Left/Right Arrow`)
    *   Moves the cursor instantly to the start or end of the current line.
*   **Select to Beginning/End of Line:** `Shift + Home` / `Shift + End` (`Cmd + Shift + Left/Right Arrow`)
    *   Selects text from the cursor to the beginning or end of the line.
*   **Move Cursor to Beginning/End of Document:** `Ctrl + Home` / `Ctrl + End` (`Cmd + Up/Down Arrow`)
    *   Moves the cursor to the absolute beginning or end of the document.
*   **Select to Beginning/End of Document:** `Ctrl + Shift + Home` / `Ctrl + Shift + End` (`Cmd + Shift + Up/Down Arrow`)
    *   Selects text from the cursor to the beginning or end of the document.
*   **Delete One Word Left:** `Ctrl + Backspace` (`Option + Delete`)
    *   Deletes the entire word to the left of the cursor.
*   **Delete One Word Right:** `Ctrl + Delete` (`Option + Fn + Delete` or `Ctrl + D` in some apps)
    *   Deletes the entire word to the right of the cursor.
*   **Page Up/Down:** `Page Up` / `Page Down` (`Fn + Up/Down Arrow`)
    *   Scrolls up or down one screen at a time.

### III. Browser Specific

These are invaluable for web browsing.

*   **New Tab:** `Ctrl + T` (`Cmd + T`)
    *   Opens a new blank tab.
*   **Reopen Last Closed Tab:** `Ctrl + Shift + T` (`Cmd + Shift + T`)
    *   Brings back the tab you just closed (can often go back multiple tabs).
*   **Go to Specific Tab:** `Ctrl + 1` through `9` (`Cmd + 1` through `9`)
    *   Switches to the tab corresponding to the number (e.g., `Ctrl + 1` goes to the first tab).
*   **Next Tab:** `Ctrl + Tab` (`Ctrl + Tab` or `Cmd + Option + Right Arrow`)
    *   Moves to the next tab to the right.
*   **Previous Tab:** `Ctrl + Shift + Tab` (`Ctrl + Shift + Tab` or `Cmd + Option + Left Arrow`)
    *   Moves to the previous tab to the left.
*   **Go Back/Forward:** `Alt + Left/Right Arrow` (`Cmd + Left/Right Arrow`)
    *   Navigates back or forward in your browsing history.
*   **Open Link in New Tab (Background):** `Ctrl + Click` (`Cmd + Click`)
    *   Opens a link in a new tab without switching to it.
*   **Open Link in New Tab (Foreground):** `Ctrl + Shift + Click` (`Cmd + Shift + Click`)
    *   Opens a link in a new tab and immediately switches focus to it.
*   **Zoom In/Out/Reset:** `Ctrl + Plus`/`Minus`/`0` (`Cmd + Plus`/`Minus`/`0`)
    *   Zooms the page in, out, or resets to 100%.
*   **Open Downloads:** `Ctrl + J` (`Cmd + Shift + J`)
    *   Opens your browser's downloads page.
*   **Open History:** `Ctrl + H` (`Cmd + Y`)
    *   Opens your browser's history page.
*   **Open Bookmarks:** `Ctrl + D` (to bookmark page) / `Ctrl + Shift + B` (to open bookmarks bar) (`Cmd + D` / `Cmd + Shift + B`)
    *   Bookmarks the current page or toggles the bookmarks bar.
*   **Private/Incognito Window:** `Ctrl + Shift + N` (`Cmd + Shift + N`)
    *   Opens a new private browsing window.

### IV. Windows Specific

*   **File Explorer:** `Win + E`
    *   Opens File Explorer.
*   **Settings:** `Win + I`
    *   Opens the Settings app.
*   **Run Dialog:** `Win + R`
    *   Opens the Run dialog box.
*   **Snap Window:** `Win + Left/Right/Up/Down Arrow`
    *   Snaps the active window to a side or corner of the screen.
*   **Virtual Desktop (New):** `Win + Ctrl + D`
    *   Creates a new virtual desktop.
*   **Virtual Desktop (Switch):** `Win + Ctrl + Left/Right Arrow`
    *   Switches between virtual desktops.
*   **Task View (Virtual Desktops):** `Win + Tab`
    *   Opens Task View, showing open windows and virtual desktops.
*   **Project Display:** `Win + P`
    *   Opens the project menu to manage multiple displays.
*   **Action Center:** `Win + A`
    *   Opens the Action Center for notifications and quick settings.
*   **Emoji Picker:** `Win + .` (period) or `Win + ;` (semicolon)
    *   Opens the emoji keyboard.

### V. macOS Specific

*   **Finder (Go to Folder):** `Cmd + Shift + G`
    *   Opens a dialog to go directly to a folder path.
*   **Hide Current Application:** `Cmd + H`
    *   Hides the current application's windows.
*   **Hide All Other Applications:** `Cmd + Option + H`
    *   Hides all applications except the frontmost one.
*   **Spotlight Search:** `Cmd + Space`
    *   Opens Spotlight search (universal search for files, apps, definitions, etc.).
*   **Quick Look:** `Spacebar` (on a selected file)
    *   Opens a preview of a selected file without opening the associated application.
*   **Mission Control:** `F3` or `Control + Up Arrow`
    *   Shows all open windows, desktops, and full-screen apps.
*   **Show Desktop:** `F11` or `Cmd + F3`
    *   Moves all windows aside to show the desktop.
*   **Dictation:** `Fn + Fn` (double press)
    *   Starts dictation (if enabled).

---

### VI. Other Useful Keys

*   **Tab:** Moves focus to the next interactive element (button, link, form field).
*   **Shift + Tab:** Moves focus to the previous interactive element.
*   **Escape (Esc):** Closes menus, dialogs, cancels operations, exits full-screen mode.
*   **Home/End:** Navigates to the beginning/end of a line (text) or list (files).
*   **Delete:** Deletes selected items.
*   **Backspace:** Deletes characters to the left of the cursor (text) or goes back in a file explorer/browser.
